# Quickipedia

Sometimes you don't have time to through an entire Wikipedia article, or, let's be honest, even the summary. Quickipedia summarizes the summary!

## Technologies

- Flask
- Wikipedia
- Algorithmia
- React.js
- Node.js
- Gulp
- Browserify
